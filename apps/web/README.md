Next.js-app voor sedaiia.com (pilot). Zelfde patronen als CreativeLeads-producten, maar deze repo is alleen de Sedaiia-site.

## Structuur

- `src/app/` App Router
- `src/components/` site-specifieke componenten
- `public/` statische assets
